program 
./dist/program.exe lub ./program.py
dokonuje konwersji między plikami: 
.yaml .xml .json

wywołanie z konsoli: 
python program.py filepath.x filepath2.y
program.exe filepath.x filepath2.y
gdzie x i y to rozszerzenia z trzech możliwych

przykładowe pliki: 
xml.xml , yaml.yaml , json.json
znajdują się w folderze razem z wykonywalną aplikacją exe
